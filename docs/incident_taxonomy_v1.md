# Incident Taxonomy v1

Simple v1 taxonomy for consistent classification and alerting.

## Incident Types (8)

- `data_breach`
- `ransomware`
- `account_takeover`
- `service_disruption`
- `vulnerability_exploitation`
- `insider_incident`
- `third_party_compromise`
- `other_security_incident`

## Severity Levels (4)

- `low`
- `medium`
- `high`
- `critical`

## Lifecycle States (4)

- `new`
- `investigating`
- `confirmed`
- `resolved`

## Confidence Bands (4)

- `low` (0.00-0.39)
- `medium` (0.40-0.69)
- `high` (0.70-0.89)
- `very_high` (0.90-1.00)
