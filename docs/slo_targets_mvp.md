# MVP SLO Targets

- Incident-to-organization precision: `>= 85%`
- Alert latency for high-confidence incidents: `<= 6 hours`
- Duplicate incident rate: `<= 8%` over rolling 7 days
- Ingestion freshness lag p95: `<= 4 hours` for Wave 1 sources
- Connector run success rate: `>= 98%` daily (excluding provider outages)
- False alert rate: `<= 10%` over rolling 14 days
